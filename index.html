About the YSWS

Make a CSS animation! Use only CSS to create a cool animation and get some art supplies.

How to Participate

  1. Design a creative CSS animation (check criteria for what counts as creative)
  2. Upload your CSS animation to GitHub, then deploy it to your platform of choice (GitHub Pages is recommended)
  3. Submit your GitHub repo and live site link to the submission form here by the deadline
  4. If you're approved, we'll be in touch to arrange the shipping of your choice between art supplies and Procreate

Judging Criteria

  • Creativity: How creative the animation is
  • Visual Appeal: The animation should look good.
  • Elements: Must use 2+ elements like text, shapes and colors
